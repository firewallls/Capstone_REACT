# Study Flow

A modern student productivity dashboard built with React, React Router, Tailwind CSS, and shadcn/ui.

Study Flow helps students manage attendance, marks, platforms, schedules, and simulations through a clean and interactive dashboard interface.

---

# Features

## Dashboard Overview

* Clean and modern student dashboard
* Responsive layout for desktop and mobile
* Interactive cards and analytics
* Sidebar navigation
* Dark and Light theme support

## Attendance Management

* Track subject attendance
* Monitor attendance percentage
* View progress visually
* Attendance insights and calculations

## Marks Tracking

* Store and manage marks
* Track academic performance
* Easy-to-read progress indicators

## Platforms Section

* Manage learning platforms and resources
* Organized dashboard view

## Simulator Panel

* Interactive simulator components
* Dynamic UI updates
* Beginner-friendly React implementation

## Theme System

* Light mode
* Dark mode
* Instant theme switching

# Installation

## 1. Clone the Repository

```bash
git clone <repository-url>
```

## 2. Open the Project Folder

```bash
cd Study_Flow/client
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Start Development Server

```bash
npm run dev
```
